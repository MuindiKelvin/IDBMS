How to run the IDBMS Project using PHP and MySQL

1. Download the  zip file

2. Extract the file and copy IDBMS folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name inventory

6. Import inventory.sql file

7.Run the script http://localhost/IDBMS
